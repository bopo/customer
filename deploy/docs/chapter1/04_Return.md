返回说明
====

#### 正常返回


> 正常返回的状态码为20x，也就是2开头的。

#### 错误返回


> 一般判断状态码就可以知道是成功还是失败。

> 一般都会返回一个json，提示内容会在`errors`键中。

例如：

```
{
    "errors": {
        "code": 40x,
        "msgs": "你要找的内容不存在。"
    }
}
```

#### 权限错误

一般返回状态码 403，401

```
{
    "errors": {
        "code": 403,
        "msgs": "身份认证信息未提供。"
    }
}
```
